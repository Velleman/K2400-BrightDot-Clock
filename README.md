https://github.com/Makuna/NeoPixelBus

ESPAsyncWebServer
https://github.com/me-no-dev/ESPAsyncWebServer

ESPAsyncTCP
https://github.com/me-no-dev/AsyncTCP

ArduinoJson
https://github.com/bblanchon/ArduinoJson
